# Custom PrintScreen

A lightweight screenshot tool with a floating capture button and customizable JPEG output.

Built for fast workflows and controlled file saving.

---

## ✨ Features

- Floating capture button
- Save directly to selected folder
- Adjustable JPEG quality
- Simple and clean interface
- No installation required
